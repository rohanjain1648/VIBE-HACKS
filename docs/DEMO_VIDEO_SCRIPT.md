## Production Notes

### Recording Setup

**Screen Recording**:
- Use OBS Studio or similar
- 1920x1080 resolution
- 60 FPS
- High quality settings

**Audio Recording**:
- Clear microphone (USB condenser recommended)
- Quiet environment
- Separate audio track for easy editing

**Browser Setup**:
- Chrome (best Web Speech API support)
- Full screen mode (F11)
- Hide bookmarks bar
- Clear browser cache
- Disable extensions that might interfere

- [ ] Trim dead space
